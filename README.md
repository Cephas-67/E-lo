## Exécution du projet en local

1. **Cloner le dépôt**
    ```bash
    git clone <URL_DU_DEPOT>
   
    ```

2. **Installer les dépendances**
    ```bash
    npm install
    ```
    ou
    ```bash
    yarn install
    ```

3. **Lancer le projet**
    ```bash
    npm run dev
    ```
    ou
    ```bash
    yarn start
    ```

> Remplacez `<URL_DU_DEPOT>` et `<NOM_DU_REPERTOIRE>` par les valeurs appropriées.
